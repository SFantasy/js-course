# React 实战

这是一本关于 React 的小书，主要内容为 React 方面的编程实践。

欢迎开始阅读：[简介](./intro/README.md)。

## 为什么写作本书？

1. 通过示例方便初学者了解 React 的世界，减少学习过程中的坑坑洼洼
2. 总结自己的一些学习心得、体会和经验
